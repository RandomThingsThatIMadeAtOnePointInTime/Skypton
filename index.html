Skypton

An extensible, plugin-driven Skype bot

Download as .zip Download as .tar.gz View on GitHub

Usage

Open Skypton.exe and check your running Skype client. On the top should be a message for Skypton requesting access to use Skype. Accept this message. Skypton is now accepting messages and has shown you all of the loaded plugins in the console. To close, just close the window.

Plugins

By default Skypton builds all of the plugins in the solution but once all of them are built you can pick and choose which plugins you want to have by deleting the plugin's .dll from the plugins folder
